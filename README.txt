# Form 1 Science Performance Analytics System

## Overview
A Streamlit-based dashboard built to track student performance in Form 1 Science classes.

## Features
- Track student scores by topic
- Calculate averages automatically
- Identify weak students
- Show student ranking
- Provide class-level insights

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run

1. Install dependencies:
pip install streamlit pandas

2. Run the app:
streamlit run app.py
